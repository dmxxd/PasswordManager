# PasswordManager

**PasswordManager** — настольное приложение для хранения и управления пользовательскими паролями и учетными данными.  
Проект предназначен для удобного и безопасного хранения информации о различных аккаунтах в одном интерфейсе.

Проект выполнен в рамках учебной практики.

---

## Зачем нужен проект

PasswordManager создан для упрощения работы с большим количеством учетных записей и паролей.

Приложение помогает:

* хранить все логины и пароли в одном месте;
* быстро находить нужные данные;
* уменьшить риск потери информации;
* упростить управление учетными записями для различных сервисов.

---

## Возможности

* добавление новых записей (сервис, логин, пароль);
* просмотр списка сохранённых учетных данных;
* редактирование существующих записей;
* удаление учетных записей;
* локальное хранение данных;
* простой и понятный пользовательский интерфейс.

---

## Технологии

* Язык: C#
* Платформа: .NET Framework / .NET
* Тип приложения: настольное (Windows Forms)
* Среда разработки: Visual Studio Community 2022
* Хранение данных: Локальная база данных или файловое хранилище

---

## Установка и запуск

1. Скачайте репозиторий или архив с проектом.
2. Распакуйте проект в удобную папку.
3. Откройте файл решения **PasswordManager.sln** в Visual Studio 2022.
4. Дождитесь восстановления зависимостей (если используется NuGet).
5. Запустите проект, нажав **F5** или через меню *Отладка → Начать отладку*.

---

## Использование

После запуска приложения пользователь может:

* добавлять новые учетные данные;
* просматривать сохранённые записи;
* изменять информацию;
* удалять ненужные данные;
* управлять всеми своими паролями в одном окне.
