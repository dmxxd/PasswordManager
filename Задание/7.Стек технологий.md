# Стек технологий

## 1. Язык программирования

**C#** выбран в качестве основного языка разработки по следующим причинам:
- является стандартным и наиболее распространённым языком для разработки десктопных приложений под Windows;
- полностью поддерживает технологию Windows Forms;
- предоставляет мощные встроенные средства для работы с безопасностью и криптографией через пространство имён `System.Security.Cryptography`;
- обладает строгой типизацией и развитой экосистемой, что снижает вероятность ошибок и повышает надёжность приложения.

---

## 2. Фреймворк

В качестве платформы разработки используется **Windows Forms (.NET Framework 4.8)**.

Обоснование выбора:
- простота разработки графического пользовательского интерфейса;
- отсутствие необходимости установки дополнительных runtime-компонентов на большинстве современных систем Windows;
- хорошая совместимость с Windows 10 и Windows 11;
- подходит для создания локальных десктопных приложений с минимальными системными требованиями.

---

## 3. Хранение данных

В системе не используется система управления базами данных (СУБД).

Вместо этого применяется:
- **локальный JSON-файл**, содержащий записи паролей и категории;
- полное шифрование файла с использованием алгоритма AES-256.

Для сериализации и десериализации данных используется библиотека **Newtonsoft.Json**, так как она:
- проста в использовании;
- широко распространена и хорошо документирована;
- обеспечивает стабильную и быструю работу с JSON-структурами.

Такой подход позволяет избежать накладных расходов на использование СУБД и упростить развертывание приложения.

---

## 4. Инструменты тестирования

Для проверки корректности работы программного обеспечения используются:
- тест-кейсы, основанные на сценариях использования системы;
- ручное функциональное тестирование основных возможностей приложения.

Данный подход является достаточным для небольшого локального приложения и позволяет проверить корректность работы всех ключевых функций.

---

## 5. Вывод

Выбранный стек технологий обеспечивает необходимый уровень безопасности, производительности и удобства разработки, а также полностью соответствует требованиям к программному продукту «Менеджер паролей с локальным шифрованием».
