# Анализ предметной области

## Менеджер паролей с локальным шифрованием

Менеджер паролей с локальным шифрованием — это приложение для безопасного хранения, управления и генерации паролей. Оно ориентировано на индивидуальных пользователей, желающих хранить учетные данные (логины, пароли) **локально на устройстве**, без передачи данных в облако, с акцентом на **криптографическую защиту** от несанкционированного доступа.

Дополнительные функции включают:
- генератор надежных паролей;
- категоризацию записей (по сайтам, приложениям или типам аккаунтов).

---

## Ключевые процессы

### Хранение паролей
Пользователь может добавлять, редактировать и удалять записи о паролях. Все данные **шифруются локально** (например, с использованием алгоритма AES) и сохраняются в зашифрованном файле на устройстве пользователя.

### Генерация паролей
Приложение автоматически создает надежные пароли на основе заданных параметров:
- длина пароля;
- использование специальных символов;
- наличие цифр;
- использование букв в верхнем и нижнем регистре.

### Категоризация
Записи группируются по категориям (например: *«Социальные сети»*, *«Банки»*, *«Работа»*), что упрощает навигацию и поиск нужных учетных данных.

### Доступ и аутентификация
Для разблокировки приложения используется **мастер‑пароль**. Без его ввода доступ к данным невозможен, а содержимое базы остается зашифрованным.

### Поиск и просмотр
Реализованы:
- поиск записей по ключевым словам;
- фильтрация по категориям;
- копирование пароля в буфер обмена с таймером автоматической очистки.

---

## Сущности предметной области

### Пользователь (актор)
Основной пользователь приложения, осуществляющий управление паролями.

### Запись пароля
Сущность, содержащая следующие поля:
- категория;
- название (например, сайт или сервис);
- логин;
- пароль (в зашифрованном виде);
- заметки.

### Мастер‑пароль
Основной ключ, используемый для шифрования и дешифрования локальной базы данных.

### База данных
Локальный файл (например, в формате JSON или XML), содержащий зашифрованные записи паролей.

### Генератор паролей
Модуль, отвечающий за создание случайных и криптографически стойких строк паролей.

---

## Требования к системе

- локальное шифрование данных для предотвращения утечек информации;
- отсутствие сетевых подключений и передачи данных по сети;
- интуитивно понятный интерфейс для быстрого доступа к паролям;
- генерация сильных паролей длиной **не менее 12 символов** с использованием букв разных регистров, цифр и специальных символов.
