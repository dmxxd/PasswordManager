# Техническое задание

---

## 1. Цели разработки

Целью разработки является разработка десктопного приложения для операционной системы Windows, обеспечивающего безопасное хранение и удобное управление пользовательскими паролями.

Основные цели системы включают:

- разработку десктопного приложения для безопасного управления паролями на платформе Windows;
- обеспечение локального шифрования данных для защиты от несанкционированного доступа;
- реализацию генератора паролей для создания сильных и устойчивых комбинаций;
- поддержку категоризации записей для удобной организации и быстрого поиска данных.

---

## 2. Описание системы

Разрабатываемая система представляет собой десктопное приложение, созданное с использованием технологии **Windows Forms** на языке программирования **C#** и платформе **.NET Framework 4.8**.

Основные характеристики системы:

- все пользовательские данные хранятся в зашифрованном файле на локальном диске;
- доступ к данным осуществляется только после ввода мастер-пароля при запуске приложения;
- пользовательский интерфейс включает формы для:
  - добавления и редактирования записей;
  - просмотра и поиска сохранённых паролей;
  - генерации надёжных паролей;
  - управления категориями;
- защита данных реализуется с использованием библиотеки **System.Security.Cryptography** и алгоритма шифрования **AES-256**;
- приложение функционирует автономно и не использует сетевые подключения или внешние сервисы.

---

## 3. Целевая аудитория

Программный продукт ориентирован на следующие группы пользователей:

- индивидуальные пользователи Windows в возрасте от 18 лет, заинтересованные в защите персональных данных;
- IT-специалисты и фрилансеры, работающие с большим количеством учётных записей;
- обычные пользователи, имеющие множество аккаунтов в различных сервисах;
- пользователи, предпочитающие локальное хранение конфиденциальной информации вместо облачных решений по соображениям приватности.

---

## 4. Общие требования

Данный раздел определяет обязательные требования к условиям эксплуатации и характеристикам программного продукта.

- язык пользовательского интерфейса: **русский**;
- минимальные системные требования:
  - операционная система: Windows 10 и выше;
  - платформа: .NET Framework 4.8 и выше;
- размер установленного приложения: **не более 10 МБ**;
- приложение должно корректно функционировать без подключения к сети Интернет.
